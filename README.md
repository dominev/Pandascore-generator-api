# Генератор сайта киберспортивных матчей (pandascore_api)

Проект автоматически создаёт трёхстраничный статический сайт с информацией о киберспортивных матчах по трем периодам: вчера, сегодня, завтра.

Данные берутся из API [pandascore](https://www.pandascore.co/).

*При каждом запуске программа обновляет страницы актуальными данными.*

## Итоговый вывод

- **3 HTML-страницы**:  
  `yesterday.html` – матчи за вчера  
  `today.html` – матчи сегодня  
  `tomorrow.html` – матчи на завтра  
- **Автоматическая навигация** между днями (ссылки вверху страницы).  
- **SEO-оптимизация**: мета-теги title, description.  
- **Микроразметка schema.org** для улучшения видимости в поисковиках.  
- **Простой и чистый дизайн**, который адаптируется под экраны компьютеров и телефонов.

## Архитектура
```bash
pandascore_generator/
├── .env                 #PANDASCORE_API_KEY
├── generate.py          # Главный скрипт
├── templates/
│   └── page.html        # Базовый шаблон HTML
├── output/              # Папка для вывода
│   ├── yesterday.html
│   ├── today.html
│   ├── tomorrow.html
├── screenshots/         # Тест скрины   
└── requirements.txt     # Зависимости
```

## Как это работает

Программа обращается к API pandascore, получает список матчей на нужные даты, форматирует их и вставляет в HTML-шаблон. Готовые файлы сохраняются в папку `output`.

## Что нужно для запуска

1. **Pyhon 3.11** 
2. **API - ключ pandascore.**  
3. **Отредактировать .env под свой API ключ.**

## Пошаговая инструкция

### 1. Скачать проект
- ```bash
    git clone https://github.com/dominev/pandascore-generator-api.git
    cd pandascore-generator-api
    ```
- Либо скачать zip - архив с репозитория напрямую.

### 2. Получить API-ключ
- [pandascore](https://www.pandascore.co/) 
- [api](https://www.pandascore.co/pricing/)   
- Скопировать ключ.

### 3. Настроить ключ в проекте
- В папке с проектом отредактировать файл с именем **.env**  
- Заменить ключ на свой в "PANDASCORE_API_KEY=XXXX(ваш_ключ)"  

### 4. Установите необходимые библиотеки

```bash
    pip install -r requirements.txt
```

### 5. Запустите генерацию сайта

```bash
    python generate.py
``` 
- Появятся сообщения о создании страниц. ОК

### 6. Проверьте результат

- В папке `output` три файла html.// открыть в браузере.
- По ссылкам «Вчера» и «Завтра» можно переключаться между днями.  
- Если на какой-то день матчей нет, то увидите сообщение об этом.
